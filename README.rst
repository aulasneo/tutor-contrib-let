let plugin for `Tutor <https://docs.tutor.edly.io>`__
#####################################################

Tutor plugin to simplify the configuration of Open edX settings.


Installation
************

.. code-block:: bash

    pip install git+https://github.com/aulasneo/tutor-contrib-let

Usage
*****

.. code-block:: bash

    tutor plugins enable let
